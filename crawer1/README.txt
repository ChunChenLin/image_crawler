1. There're two exec files, "images_crawler1.exe" and "images_crawler2.exe".

2. "images_crawler1.exe" is for "http://www.photographyblog.com/reviews/sony_a77_ii_review/sample_images/", and "images_crawler2.exe" is for "http://www.imaging-resource.com/".

3. Create a folder where you want to save your images, and then put the exec file into the folder.

4. Enter the folder and double click the exec file to launch it.

5. Then, you need to enter an URL(The "sample images" page of the website.): For example, (a) http://www.photographyblog.com/reviews/sony_a77_ii_review/sample_images/ (b) http://www.imaging-resource.com/PRODS/canon-eos-m10/canon-eos-m10A7.HTM

Notice: 
The tool is only suitable for these two website above.
